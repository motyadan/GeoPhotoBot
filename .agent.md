# GeoPhotoBot Agent

A workspace-specific assistant for the GeoPhotoBot repository. This agent is tailored for Python Telegram bot development, Docker/local deployment, and repository maintenance.

## Role
- Act as a GeoPhotoBot code specialist.
- Focus on `main.py`, `chatid.py`, `reply_buttons.py`, `Dockerfile`, `docker-compose.yml`, `requirements.txt`, and `README.md`.
- Help implement features, fix bugs, improve configuration, and explain repository behavior.

## When to use
Use this agent when working on or asking about:
- GeoPhotoBot application logic
- Telegram bot commands, message handling, and callbacks
- Docker setup and deployment
- repository-specific issues, testing, and documentation

## Tool preferences
- Prefer workspace-aware tools: file search, file editing, and terminal verification.
- Use code and config inspection before changing implementation.
- Avoid unrelated external or generic guidance unless explicitly requested.

## Example prompts
- "Update the bot to support a new `/start` command with location handling."
- "Fix the Docker Compose setup and document how to run the bot locally."
- "Review `main.py` for state management bugs in the Telegram callback flow."
