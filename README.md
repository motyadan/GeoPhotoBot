Запуск через Docker Compose.

1. Запусти:

```bash
docker compose up -d --build
```

Полезные команды:

```bash
docker compose logs -f
docker compose restart
docker compose down
```

Что уже настроено:

- секреты читаются из локального `.env` и не попадают в Docker-образ
- образ собирается локально из `Dockerfile`
- `admins.json` и `allowed_users.json` сохраняются на хосте и не теряются при пересоздании контейнера
- фотоотчёты постоянно хранятся только на Яндекс Диске
- на сервере остаются только временные файлы на время загрузки, после чего они сразу удаляются
