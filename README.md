Запуск через Docker Compose.

1. Скопируй `.env.example` в `.env`.
2. Заполни в `.env` все токены, ID и ключи.
3. Запусти:

```bash
docker compose up -d
```

Фотоотчёты постоянно хранятся только на Яндекс Диске. На сервере используются лишь временные файлы на время загрузки, после чего они сразу удаляются.
